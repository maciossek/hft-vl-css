CSS

CSS1 (Dec. 1996)

  • Schriftarten und „Betonung setzen“
  • Farben für Schrift, Hintergrund und Elemente
  • Text-Attribute (wie Wortabstand, Zeichengrößen und Textlinien)
  • Alignment von Text, Bildern, Tabellen und anderen Elementen
  • Randabstand, Ränder, Einrückung und Positionierung
  • Eindeutige Identifikation von Gruppen von Attributen

CSS2 (May 1998)

  • Aufbauend auf CSS1
  • Erweiterungen um absolute und relative, sowie feste Positionen
  • Z-Index (Tiefeninformation)
  • „Aural Style Sheets“ für Sprachsynthese CSS Aural Reference

CSS2.1 (Feb. 2004, dann zurückgezogen, Final Jun. 2011)

  • Ausmerzen von Fehlern in CSS2, mit „Rückwärtskompatibilität“

CSS3 (Releases seit Juni 2011 bis Juni 2012)

  • Standardunterstützung nun basierend auf Levels, hier Level3
  • Wieder aufbauend auf CSS2 -- nun aber modulare Einzeldokumente:
    • Juni 2011: CSS Color Module Level 3
    • Sept. 2011: Selectors Level 3
    • Sept. 2011: CSS Namespace Module
    • Jun. 2012: Media Queries

„CSS4“ (Ongoing)

  • Level 4 ist im Aufbau begriffen -- „There will never be a CSS4.“
  • Weitere Module (hier die finalisierten):
    • Apr. 2012: Requirements for Japanese Text Layout
    • Sept. 2012: W3C Working Draft „Image Values“
    • Mar. 2013: CSS Print Profile
    • Oct. 2013: Selectors API Level 2

Warum gibt es CSS?

  • Trennung von Layout und Inhalt
  • Wiederverwendbarkeit
  • Verbesserte Anpassungen möglich
  • Bandbreite
Bezüglich Bandbreite: Einmal übertragene Daten / Stildefinitionen für referenzierte CLASS werden nur einmal übertragen.
